# react-native-openvpn

Open source react native library for integrating OpenVPN into React Native.

## Installation


```sh
npm install react-native-openvpn
```


## Usage


```js
import { multiply } from 'react-native-openvpn';

// ...

const result = multiply(3, 7);
```
